# ⚡ WordPress 一键备份还原（高性能版）

一个面向实际生产环境的 WordPress 备份与还原工具。  
核心目标：**速度优先、稳定优先、可控优先**。

> ⚠️ 本项目为自用优化版本，不追求极端兼容性。

---

## 🚀 核心特性

### ⚡ 高性能还原
- 使用系统命令执行（`exec / popen`）
- 数据库直导（避免 PHP 逐条执行 SQL）
- 文件原地覆盖（避免低效 `copy`）
- 分段执行（规避超时限制）
- 支持大站点恢复

---

### 📦 一键备份
- 数据库 + 文件完整打包
- 单文件输出（`.bgbk`）
- 自动生成元信息
- 支持大体积站点

---

### 🔄 分段还原引擎
- 异步执行（避免 Cloudflare 100 秒限制）
- 自动分阶段执行：
  - 解压
  - 文件恢复
  - 数据库导入
- 支持进度追踪

---

### 🧠 环境检测机制
- 自动检测运行环境
- 不满足条件直接终止执行
- 避免执行过程中失败

## ⚡ 性能说明

相比常规 WordPress 备份插件：

| 项目 | 本工具 | 通用插件 |
|------|--------|----------|
| 数据库导入 | 系统级导入 | PHP 执行 |
| 文件处理 | 原生覆盖 | PHP copy |
| 执行方式 | 分段任务 | 多次请求 |
| 性能 | ⚡ 高 | 🐢 较低 |

说明：

通用插件为了兼容性，通常使用纯 PHP 实现，  
而本工具直接使用系统能力，减少中间层开销。

### ❓ 还原速度慢
可能原因：

- 磁盘 IO 瓶颈
- MySQL 写入性能限制
- 未使用系统级导入

### ❓ 进度异常 / 秒完成

可能原因：

- AJAX 未正确执行
- 环境检测未通过
